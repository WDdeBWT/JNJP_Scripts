项目简介

本项目针对垃圾焚烧“邻避”事件这一热点问题，对国内外垃圾焚烧相关案例进行研究分析，深刻剖析“邻避”事件产生的原因，借鉴国内外“邻利”先进经验，提出化解“邻避”事件的体制、机制、模式创新，为解决“邻避”事件提供新思路，开创新途径。同时综合运用网络爬虫、NLP等技术，借助分词系统等相关工具，采集并分析大量垃圾焚烧“邻避”相关案例，构建贝叶斯网络模型，为决策者提供精准有效的技术指导。

创新点

提出化“邻避”为“邻利”的体制创新、机制创新以及模式创新方式。基于贝叶斯网络模型，运用网络爬虫、分词系统、NLP等技术，构建“名利”空间下垃圾危机时空情景演化，为决策者决议提供理论依据和技术支撑。

团队介绍

我们的团队“ISAID”是由来自武汉理工大学管理学院信息管理与信息系统专业的七位同学：周惠、汪林岚、白文涛、何睿、谢豪、杜志洪和刘恩源组成。

"I" ( innvation ) 象征着我们锐意进取，敢于创新的特点，"S" ( Statistics )、"A" ( Analysis )、"I" ( Investigation ) 和 "D" ( Decision )表示我们研究制作项目中所运用到的统计、分析、调查、决策的手段方法。我们的团队不光在论文撰写上有一定的经验，懂得基本的研究调查方法及实际的运用，作为信管专业的学生，我们更具备一定的编程能力、数据分析经验和研究，在完成社会实践项目的同时运用数据和计算机技术的支撑。在整个研究过程中，团队成员都秉承着科学严谨的态度，勇于探索、敢于创新，其中遇到了很多挫折，但是队员们都发挥各自的所长、互相帮助，不气馁不放弃，在指导老师的带领下，最终将整个项目较为完整的呈现出来。
